HTML - Básico

Categoria do Curso

Desenvolvimento de aplicativos

Carga horária

11 horas

Descrição do curso

Este curso apresenta conceitos básicos de web e mostra as partes e funções das togs em um documento HTML. Ensina, também, a formatar uma página, inserir imagens e muito mais! Tudo isso, com explicações simples e de fácil compreensão.

Estrutura do curso

  1. Módulo 1 - Introdução
  2. Módulo 2 - HTML e a web
  3. Módulo 3 - Criando um documento HTML
  4. Módulo 4 - Formando um documento HTML
  5. Módulo 5 - Criando listas
  6. Módulo 6 - Inserindo Imagens
  7. Módulo 7 - Criando links
  8. Módulo 8 - Combinando tags
  9. Módulo 9 - Usando imagens mapeadas

Pré-requisitos

Não há pré-requisitos para a realização deste curso
